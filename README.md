### Export Formats

- **SVG** - 2D vector output for laser engraving and visualization
- **STEP (.stp)** - 3D CAD format for CNC milling toolpath generation
- **STL** - 3D mesh format for CNC machining and 3D printing

## Installation

### Rust

```bash
cargo add turtle
```

### Python

